3. Table P-3, Bomb Bay, result "4" = Rubber Rafts = The Rubber Rafts are unusable, and the roll on Table G-10 when landing in water is −6.

**B-5 Walking Hits Notes:**

A "walking hits" result negates all other shell hits by this fighter for this attack and does the following damage:

- **(a)** 1 shell hit in each of: Nose, Pilot Compartment, Bomb Bay, Radio Room, Waist, and Tail.
- **(b)** 2 shell hits on each Wing.
- **(c)** 1 shell hit in each of: Nose, Wing (attacking side), Waist, and Tail.
